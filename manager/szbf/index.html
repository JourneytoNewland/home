    孙子兵法

    《孙子兵法》

    * * :scream: 不是教你诈

    善良 :sparkling_heart: 比聪明更难

    1. 计篇

    • 兵者，国之大事，死生之地，存亡之道，不可不察也
    • 五事
    • 道：令民与上同意
    • 天：阴阳、寒暑、时制
    • 地：远近、险易、广狭、死生
    • 将：智、信、仁、勇、严
    • 法：曲制、官道、主用
    • 七计
    • 主孰有道
    • 将孰有能
    • 天地孰得
    • 法令孰行
    • 兵众孰强
    • 士卒孰练
    • 赏罚孰明
    • 计利以听，乃为之势，以佐其外。势者，因利而制权也
    • 庙算之重：多算胜，少算不胜，无算滚犊子

    2. 作战篇

    • 战前兵马粮等需充分准备，不可轻易出兵
    • 不尽知用兵之害者，则不能尽知用兵之利
    • 善用兵者
    • 役不再籍，粮不三载
    • 取用于国，因粮于敌
    • 车杂而乘之，卒善而养之，是谓胜敌而益强
    • 兵贵胜，不贵久

    3. 谋攻篇

    • 百战百胜，非善之善者也；不战而屈人之兵，善之善者也
    • 必以全争于天下
    • 上兵伐谋，其次伐交，其次伐兵，其下攻城。攻城之法，为不得己
    • 不战而屈人之兵
    • 用兵之法，十则围之，五则攻之，倍则分之，敌则能战之，少则能逃之，不若则能避之
    • 患于军者三
    • 不知军之不可以进而谓之进，不知军之不可以退而谓之退
    • 不知三军之事而同三军之政
    • 不知三军之权而同三军之任
    • 知胜有五
    • 知可以战与不可以战者胜
    • 识众寡之用者胜
    • 上下同欲者胜
    • 以虞待不虞者胜
    • 将能而君不御者胜
    • 知彼知己，百战不殆；不知彼而知己，一胜一负；不知彼而知己，每战必殆

    4. 军形篇

    • 善战者
    • 先为不可胜，以待敌之可胜
    • 能为不可胜，不能使敌之必可胜
    • 自保而全胜
    • 不可胜者，守也，可胜者，攻也
    • 守者不足，攻者有余
    • 善守者，藏于九地之下；善攻者，动于九天之上
    • 胜兵先胜而后求战，败兵先战而后求胜
    • 立于不败之地，而不失敌之败也
    • 战胜不忒
    • 修道而保法
    • 一曰度，二曰量，三曰数，四曰称，五曰胜。地生度，度生量，量生数，数生称，称生胜
    • 胜兵若以镒称铢，败兵若以铢称镒

    5. 兵势篇

    • 用兵原则
    • 分数
    • 形名
    • 奇正
    • 虚实
    • 凡战者，以正合，以奇胜
    • 激水之疾，至于漂石者，势也；鸷鸟之疾，至于毁折者，节也
    • 善战者，其势险，其节短
    • 善战者，求之于势，不责于人，故能择人而任势

    6. 虚实篇

    • 善战者致人而不致于人
    • 敌佚能劳之，饱能饥之，安能动之。出其所不趋，趋其所不意。行千里而不劳者，行于无人之地也
    • 善攻者，敌不知其所守；善守者，敌不知其所攻
    • 备前则后寡，备后则前寡，备左则右寡，备右则左寡，无所不备，则无所不寡
    • 侦察虚实步骤要决
    • 策之而知得失之计
    • 作之而知动静之理
    • 形之而知生死之地
    • 角之而知有余不足之处
    • 兵形象水，水之形，避高而趋下，兵之形，避实而击虚
    • 兵无常势，水无常形，能因敌变化而取胜

    7. 军争篇

    • 军争之难者，以迂为直，以患为利
    • 迂其途，而诱之以利，后人发，先人至
    • 举军而争利则不及，委军而争利则辎重捐
    • 兵以诈立，以利动，以分和为变
    • 不知诸侯之谋者，不能豫交
    • 不知山林、险阻、沮泽之形者，不能行军
    • 不用乡导者，不能得地利
    • 先知迂直之计者胜
    • 其疾如风，其徐如林，侵掠如火，不动如山，难知如阴，动如雷震
    • 掠乡分众，廓地分利，悬权而动
    • 人既专一，则勇者不得独进，怯者不得独退，此用众之法
    • 三军可以夺气，将军可夺心
    • 治气：朝气锐，昼气惰，暮气归。善用兵者，避其锐气，击其惰归
    • 治心：以治待乱，以静待哗
    • 治力：以近待远，以佚待劳，以饱待饥
    • 治变：无邀正正之旗，无击堂堂之陈
    • 用兵之法
    • 高陵勿向
    • 背丘勿逆
    • 佯北勿从
    • 锐卒勿攻
    • 饵兵勿食
    • 归师勿遏
    • 围师遗阙
    • 穷寇勿迫

    8. 九变篇

    • 将通于九变之地利
    • 圮地无舍，衢地合交，绝地无留，围地则谋，死地则战。
    • 涂有所不由，军有所不击，城有所不攻，地有所不争，君命有所不受
    • 智者之虑，必杂于利害
    • 杂于利，而务可信也
    • 杂于害，而患可解也
    • 无恃其不来，恃吾有以待也；无恃其不攻，恃吾有所不可攻也
    • 将有五危，不可不察
    • 必死可杀
    • 必生可虏
    • 忿速可侮
    • 廉洁可辱
    • 爱民可烦

    9. 行军篇

    • 四军之利
    • 处山之军：绝山依谷，视生处高，战隆无登
    • 处水上之军：无附于水而迎客，视生处高，无迎水流
    • 处斥泽之军：交军于斥泽之中，必依水草而背众树
    • 平陆之军：平陆处易，而右背高，前死后生
    • 丘陵堤防，必处其阳，而右背之
    • 凡地有绝涧、天井、天牢、天罗、天陷、天隙，必亟去之，勿近
    • 伏奸之所处：险阻、潢井、葭苇、山林、蘙荟，必谨覆索之
    • 侦察现象
    • 众树动：来
    • 众草多障：疑
    • 鸟起：伏
    • 兽骇：覆
    • 尘高而锐：车来
    • 卑而广：徒来
    • 散而条达：樵采
    • 少而往来：营军
    • 战时情况
    • 辞卑而益备，进
    • 辞强而进驱，退
    • 轻车先出居其侧，陈
    • 无约而请和，谋
    • 奔走而陈兵车，期
    • 半进半退，诱
    • 兵非益多也，惟无武进，足以并力、料敌、取人而已
    • 令之以文，齐之以武。令素行以教其民

    10. 地形篇

    • 地形有六，不可不察
    • 通：我可以往，彼可以来。先居高阳，利粮道
    • 挂：可以往，难以返。敌无备，出而胜之；敌若有备，出而不胜，难以返
    • 支：出而不利，彼出而不利。引而去之，令敌半出而击之
    • 隘：我先居之，必盈之以待敌；若敌先居之，盈而勿从，不盈而从之
    • 险：我先居之，必居高阳以待敌；若敌先居之，引而去之，勿从
    • 远：势均，难以挑战，战而不利
    • 此六者，败之道
    • 走兵：势均，以一击十
    • 弛兵：卒强吏弱
    • 陷兵：吏强卒弱
    • 崩兵：大吏怒而不服，遇敌怼而自战，将不知其能
    • 乱兵：弱不严，教道不明，吏卒无常，陈兵纵横
    • 北兵：将不能料敌，以少合众，以弱击强，兵无选锋
    • 地形，兵之助。料敌制胜，计险厄远近，知此而用战者必胜，不知此而用战者必败。
    • 视卒如婴儿，故可与之赴深溪；视卒如爱子，故可与之俱死
    • 知天知地，胜乃不穷
    • 知吾卒之可以击，而不知敌之不可击，胜之半
    • 知敌之可击，而不知吾卒之不可以击，胜之半
    • 知敌之可击，知吾卒之可以击，而不知地形之不可以战，胜之半

    11. 九地篇

    • 九地
    • 散地：无战。一其志
    • 轻地：无止。使之属
    • 争地：无攻。趋其后
    • 交地：无绝。谨其守
    • 衢地：合交。固其结
    • 重地：掠。继其食
    • 圮地：行。进其涂
    • 围地：谋。塞其阙
    • 死地：战。示之以不活
    • 兵之情主速，乘人之不及，由不虞之道，攻其所不戒也
    • 善用兵者，譬如率然；率然者，常山之蛇也。击其首则尾至，击其尾则首至，击其中则首尾俱至
    • 投之亡地然后存，陷之死地然后生

    12. 火攻篇

    • 火攻有五
    • 火人
    • 火积
    • 火辎
    • 火库
    • 火队
    • 五火之变
    • 火发于内，则早应之于外
    • 火发兵静者，待而勿攻，极其火力，可从而从之，不可从而止
    • 火可发于外，无待于内，以时发之
    • 火发上风，无攻下风
    • 昼风久，夜风止
    • 非利不动，非得不用，非危不战

    13. 用间篇

    • 先知者，不可取于鬼神，不可象于事，不可验于度，必取于人，知敌之情者也
    • 五间俱起，莫知其道
    • 因间：因其乡人而用之
    • 内间：因其官人而用之
    • 反间：因其敌间而用之
    • 死间：为诳事于外，令吾间知之，而传于敌间也
    • 生间：反报
    • 非圣智不能用间，非仁义不能使间
    • 三军之事，莫亲于间，赏莫厚于 间，事莫密于间
    • 五间之事，主必知之，知之必在于反间
    • 必索敌间之来间我者，因而利之，导而舍之，故反间可得而用，乡间、内间可得而使，死间为诳事可使告敌，生间可使如期
