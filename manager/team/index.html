    重新定义团队

    重新定义团队

    所有人都认为自己很擅长面试和评估他人，但其实只有极少的人是真正擅长的。

    职场中能够充分发挥主观能动性，具有较强执行力的人，往往能能赢得机遇，获得快速成长。很多时候，人和人的差距就在于“多想一点”、“多做一点”。

    “这妖猴能有多大神通，就敢称此名号？不要怕，吃吾一剑！”

    一、 成为一名创始人

    • 谷歌工作法则：成为一名创始人
      • 把自己看成是一名创始人
      • 像创始人一样行动

    二、给员工自由

    • 十大信条

      • 以用户为中心，其他一切自然水到渠成
      • 专心将一件事做到极致
      • 越快越好
      • 网络上也讲民主
      • 信息随时随地可得
      • 赚钱不必作恶
      • 信息无极限
      • 信息需求无国界
      • 认真不在着装
      • 追求无止境

    • 三块基石

      • 使命
      • 透明
      • 发声的权利

    • 其他

      • 世界上最有才华的人需要能够激励人心的抱负
      • 每个人都希望工作有意义
      • 如果你给人以自由，他们将还你惊喜

    • 谷歌工作法则：打造了不起的文化

      • 将工作看作是一种命运的召唤，而且工作要有富于意义的使命
      • 给人以稍多于你的舒适区的信任、自由和自主权。如果你没有感到紧张，那是因为你给的还不够

    三、只聘用比你更优秀的人

    • 谷歌工作法则：关于招聘

      • 慢慢来，聘用最优秀的人才，只聘用在某些特定的方面比你更优秀的人，不要让经理独自做团队人员聘用决策
      • 资源有限的情况下，将人力资源费用首先投入到招聘上

    • 设计有效的培训是一件困难的事，90%的培训都不能给员工表现和行为带来持续改善

    四、搜寻最优人才

    • 建立团队，坚持招聘高质量人才

    • 大幅度减少应聘者面试次数

    • 前期内推，中期数量跟不上

    • 重组招聘团队

      • 重心从筛选简历到搜寻和培养人才
      • 建立一个应聘者数据库

    • 谷歌工作法则：搜索非凡的应聘者

      • 要详细说明寻找人才的标准，依此找到最优秀的被推荐人
      • 使招聘成为每个人的工作
      • 不要害怕尝试疯狂的事情，以此引起最优秀人才的注意

    五、不要相信你的直觉

    • 根据头10秒作出的预测是没有意义的

    • 预测工作表现的方法（准确性）

      • 样例测试（以实际工作类似样例）（29%）
      • 一般认知能力测试（类似IQ测试）（26%）
      • 结构化测试（26％）
        • 行为测试（讲述过去成就）
        • 情景测试（虚拟工作场景）
      • 责任性评估（10%）
      • 背景调查（7%）
      • 工作年限（3%）

    • 面试

      • 该问哪些问题：一般认知能力／领导力／和谷歌气场符和的人／职务相关
      • 不仅要评估面试者，还要让他喜欢上你（面试后用工具搜集反馈意见）

    • 谷歌工作法则：筛选新雇员

      • 设定高质量标准
      • 寻找自己的应聘者
      • 客观评估应聘者
      • 给应聘者一个加入的理由

    六、打造最幸福的公司

    • 员工20%时间做工作之外的与公司有联系的兴趣项目

    • 决策应该在组织尽可能低的层级做出

    • 谷歌工作法则：授权于员工

      • 消除地位象征
      • 依靠数据而不是经理的想法做决定
      • 探寻方法，让员工塑造自己的工作和公司
      • 高期待获得高收益

    七、为什么每个人都讨厌绩效管理

    • 关注过程而不关注目的，使员工造假数据

    • 保证公平，增加团队经理校准环节

    • 通过2次独立谈话（绩效评估／人员发展）推动员工的进步，不要同时进行

      • 谷歌工作法则：绩效管理
        • 正确设定目标
        • 收集同事的反馈意见
        • 通过校准流程确定考评结果
        • 把奖励分配谈话与员工发展谈话分开

    八、管理团队两端－最优员工和最差员工

    • 实例：氧气项目

      • 做一名好的导师
      • 给团队授权，不随便插手下属工作
      • 表达出对团队成员的成功和个人幸福的兴趣和关心
      • 高效／结果导向型
      • 善于沟通－－聆听和分享信息
      • 在职业发展方面助力团队
      • 对团队有清晰的远景和战略
      • 具备重要的技术技能，可为团队提供建议

    • 谷歌工作法则：

      • 助力有难处的员工
        • 底端分布投入时间精力有助于团队提高
        • 要提前告知
      • 将最优秀的人放到显微镜下观察
      • 利用调查和检查清单寻找真相，推动员工学习
      • 与人分享员工对你的反馈，以身作则采取行动解决问题

    九、打造学习型组织

    • 谷歌工作法则：打造学习型组织

      • 进行刻意练习
        • 课程分成小块
        • 给出明晰反馈意见
        • 不断重复

    • 请最优秀的员工教学

    • 只对能够改变行为的课程投入

    十、不公平薪酬

    • 谷歌工作法则：激励员工并使其保持雄心

      • 不公平薪酬

        • 员工工资可偏离市场（高和低）
        • 为什么公司不设计一种体系，避免最优秀和潜力最大的员工辞职呢
          • 因为他们对公平有一种错误认识，没有勇气坦诚面对自己的员工
          • 薪酬的公平并不是说所有在同级别岗位上的人都要拿同样的薪水或是上下差不到20%
          • 因为非凡者拉动了平均数，高于中位数，所以中等水平员工要低于中等水平

      • 以成就为荣，不以报酬为荣

      • 奖励评选透明

      • 金钱奖励改为体验奖励（旅行，团队活动，等值礼品）效果更好

    • 创造易于传播爱的环境

    • 精心筹划却遭受失败的要奖励

    十一、世上最好的东西是免费的

    • 为员工提供便民商业服务，一个关联谷歌的社区

    • 图书馆，咖啡机等人员交集的地点能够推动创新

    • 谷歌工作法则：效率、社区意识和创新精神

    • 使员工的生活容易一些
    • 想办法说可以
    • 一旦员工遭遇不幸，要伸出援手

    十二、助推

    • 5美元实验（两套思维系统）

    • 29%的人愿意为125美元省下5美元（慢、深度、思索、以数据为导向）

    • 68%的人愿意为15美元省下5美元（快、依靠本能、属于直觉思维系统）

    • 利用助推使员工更开心，更高效

    • 实例

    • 每季度做个问卷（他帮助过我／我帮助过他），员工只能看到自己的结果排名

    • 新员工入职

      • 进行一次角色与责任讨论
      • 对工作期望理解不清晰的新毕业入职员工，第一年离职的概率是清晰了解工作期望的员工的5倍
      • 新员工与同事组成互帮互助组
      • 帮助新员工建立社交网络
      • 前六个月，每个月一次上岗情况检查
      • 鼓励畅所欲言

    • 其他

    • 所有人都认为自己很擅长面试和评估他人，但其实只有极少的人是真正擅长的

    • 通过学习解决

    • 谷歌工作法则：助推走向健康、富有和快乐

    • 区分“实是”和“应是”的不同

    • 进行许多小的实验
    • 助推，不要硬推

    十三、谷歌的教训

    • 透明的代价

    • 摒弃应得权利

    • 一次失败的绩效管理变革

    • 年底考核时间习惯为12月，改为3月，经过调查改为10月

    • 珍视怪人

    • 有的放矢

    • 同一时间尝试了太多东西，因为没有一件做到希望水平

    • 信念的飞跃

    • 谷歌工作法则：搞砸的时候

    • 承认错误。坦诚面对错误
    • 吸取各个方面的意见
    • 不管什么坏掉了，修好
    • 找出错误中的寓意，加以传播

    总结 十步实现团队和工作场所的改变

    • 赋予工作意义
    • 相信员工
    • 只聘用比你更优秀的人
    • 要将职业发展与管理绩效混为一谈
    • 关注团队的两端－最优员工和最差员工
    • 既要节俭又要慷慨
    • 不公平薪酬
    • 助推
